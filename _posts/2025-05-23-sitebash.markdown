description: |
  SiteDash는 최신 웹 기술 스택을 활용한 완전한 Full-stack 웹 애플리케이션입니다. 

  **주요 특징:**

  - **프론트엔드**: React 19와 TypeScript 기반의 모던한 UI/UX로 반응형 디자인 구현
  - **백엔드**: Express.js를 활용한 RESTful API 서버 구축
  - **데이터베이스**: Supabase(PostgreSQL) 클라우드 데이터베이스로 확장성 있는 데이터 관리
  - **배포 아키텍처**: 
    - 프론트엔드: GitHub Pages 자동 배포 (gh-pages 브랜치)
    - 백엔드: Railway 클라우드 플랫폼 배포
    - 데이터베이스: Supabase 클라우드 서비스

  **기술적 특징:**

  - CORS 설정을 통한 안전한 프론트엔드-백엔드 통신
  - Google OAuth 2.0 및 JWT 기반 인증 시스템 구현
  - Supabase 클라이언트를 활용한 효율적인 데이터베이스 쿼리 관리
  - 환경변수를 통한 보안성 강화
  - Zustand를 활용한 실시간 상태 관리
  - FSD(Feature-Sliced Design) 아키텍처로 확장성과 유지보수성 향상
  - Framer Motion을 활용한 인터랙티브한 UI 애니메이션
  - Sonner를 통한 사용자 친화적인 토스트 알림 시스템

  이 프로젝트는 현대적인 클라우드 네이티브 웹 개발의 모범 사례를 보여주며, 각 서비스의 장점을 최대한 활용하여 안정적이고 확장 가능한 웹 애플리케이션을 구현했습니다.
